HTML AND CSS Basics Summary

HTML Summary

HTML is used to define our page content, sructure and meaning. You don't use it for styling purposes. Use CSS for that instead

  • HTML uses elements to describe content
  • HTML elements tipically have an opening tag, content and a closing tag
  • You can also have void elements like images
  • You can also configure elements with attributes
  • There's a long list of available elements but you'll gain experiece over time

Learn more about all HTML elements on the MDN HTML element reference

CSS Summary

CSS is used for styling your page content

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you tipically use global styles
  • Or you can work with external stylesheet files
  • Inheritance, Specificity and the box model should be understood

Learn more about all CSS properties and values on the MDN CSS property reference
